Dies ist eine "Testseite" mit der das Konzept von mincss4mp aufgezeigt werden soll.

Die Testseite sollte möglichst einfach sein, aber alle angebotenen CSS-Klassen einsetzen, darstellen und dokumentieren. Für das notwendige javascript wird Vue.js verwendet. Aus Gründen der Einfachheit wird weitgehend auf Vue᠆Components verzichtet.

Die Klassen von mincss4mp bestehen, bis auf wenige Ausnahmen, nur aus Kürzeln mit Bindestrich (z.B. t᠆gr für Text᠆grau)

Horizontale Navigation

Eine horizontale Seitennavigation besteht aus einem nav-Tag und der Klasse nav und einer bestimmten Struktur (siehe Quelltext). Die eine zusätzliche Klasse nav-bk erzeugt einen schwarzen Hintergrund für die Navigation. Weitere mögliche Farben sind:

  • Klasse nav-bl= blau
  • Klasse nav-rd= rot
  • Klasse nav-gn= grün

Quelltext-Beispiel:
<nav class="nav" tabindex="-1">   <div class="nav-cont" >   <a class="pagename" href="#">HOME</a>   <a href="https://linkfoo.de/">Normaler Eintrag</a>   <div class="nav-drop">    <a href="#0" class="nav-drop-btn">Drop-Eintrag</a>    <div class="drop">       <a href="#1">Drop Eintrag 1</a>       <a href="#2">Drop Eintrag 2</a>    </div>   </div>   <button class="btn btn-out btn-nav-close">×</button>   </div> </nav>

Vertikales Menü

Eine vertikale Menüleiste besteht aus einem div-Tag und der Klasse menu und einer bestimmten Struktur (siehe Quelltext). In dieser Struktur gibt es als ersten Knoten eine a-Tag mit der Klasse drop (zeigt den Pfeil zum ausklappen), mit dem über javascript die Liste mit der Klasse menu-list aufgeklappt und eingeklappt werden kann. Analog funkioniert das mit Untermenüs. Es gibt zusätzliche Klassen für Farben:

  • Klasse nav-bl= blau
  • Klasse nav-rd= rot

Quelltext-Beispiel:
<div class="menu" >   <a class="drop" >Seiten Menü</a>   <ul id="m1" class="menu-list smooth" >      <li><a href="#m1">Punkt 1</a></li>      <li class="drop">         <a class="drop" >Unterpunkte</a>         <ul id="m2" class="menu-list smooth" >            <li><a href="#_m1">Unterpunkt 1</a></li>            <li><a href="#_m2">Unterpunkt 2</a></li>         </ul>      </li>      <li><a href="#m3">Punkt 3</a></li>   </ul> </div>

Input-Feld mit vorrangestelltem Symbol wird erzeugt mit einem <span>-Tag und der Klasse "addon".

Quelltext-Beispiel:
<span class="addon">$</span><input type="text" class="smooth" />

Ein eigens gestaltetes Input-Feld (Type=file) für Dateiauswahl mit vorrangestelltem Symbol wird erzeugt mit einem <span>-Tag und der Klasse "addon". Darauf folgt eine <output>-Tag und der Klasse "outp" und dann das normale Input-Feld, welche mit der Klasse "hidden". Durch das umschliessende <label>-Tag, wird das Event auch auf das Outputelement ausgelöst.

<label for="thefile" class="label">File<br/> <span class="addon">➡</span><output id="xthefile" type="text" class="outp" style="min-width: 228px;">{{v_file}}</output> <input id="thefile" type="file" class="hidden" v-on:change="vm_test($event,'thefile')" /> </label>

Farben

In mincc4mp werden 8 Farben verwendet (einschließlich Schwarz und Weiß). Zusätzlich gibt es drei Stufen "normal","ligth" und "dark". Um Elemente einzufärben braucht nur eine Klasse angegeben werden. Diese wird aus einzelnen Kürzeln zusammengesetzt. Als erstes das Kürzel für entweder Hintergrund (bg), Text (t) oder Umrandung (bo). Für die Farbe gibt es immer ein entsprechendes Kürzel (z.B. -bk für schwarz), welches mit einem Minuszeichen angehägt wird. Die Stufe wird mit ganz hinten angehängt (z.B. -drk)

Mit einer ähnlich benannten Hover-Klasse werden die gleichen Effekte wie bei einem Button erzeugt.

Hinweis zu Umrandungen
Eine farbige Umrandung setzt einen Rahmen vorraus. Hat das Element noch keinen Rand braucht es zusätzlich die einfache Klasse "bo" (bzw. bo2, bo3 oder bo4) welche einen Rahmen erzeugt.
Die Klassen "bo-rad0", "bo-rad3", "bo-rad5", "bo-rad7" und "bo-rad11" erzeugen abgerundete Ecken.

Tabs Info

Um Tabs darzustellen ist ein bestimmtes Konstrukt notwendig. In einem Container mit der Klasse tabs kommt als erstes ein DIV-Container mit der Klasse tab mit A-Tags für die Tabreiter. Anschließend folgen DIV-Container mit der Klasse tab-cont für den Inhalt.

Tabs sind aber unter einer bestimmten Breite nicht mehr praktikabel. Deshalb ist es ratsam vor jedem tab-cont-Container ein LABEL-Tag mit der Klasse tab-label, im LABEL ein A-Tag mit dem selben Text wie im zugehörigem Tabreiter. Die Labels werden erst ab einer Breite von unter 550px anstelle der Tabreiter eingeblendet. Für die Wechsel-Aktion ist aber javascript notwendig, hier wird vue.js verwendet. Auch die Zuordnung der Tabreiter zum Content erfolgt nur über javascript.

Farbe gibt es mit den Klassen tabs-bl(blau), tabs-rd(rot) und tabs-gn(grün), die im übergeordneten Container die Klasse tabs ersetzen. Es wird nur obere Rand des ausgewählten Tabreiters und der Schatten der Box eingefärbt.

Accordion Info

Um ein Accordion darzustellen ist ein bestimmtes Konstrukt notwendig. Immer zwei aufeinander folgende DIV-Container gruppiert in einem Container mit der Klasse acco. Der erste Container (für die Klickbereiche bzw. die Überschrift) muss die Klasse acco-label und der darauffolgende Container (für den Inhalt) acco-cont mit einem Inhalts-Tag der in allen Boxen gleich breit ist z.B. ein <P>-Tag.

Für die Wechsel-Aktion ist aber javascript notwendig, hier wird vue.js verwendet. Auch die Zuordnung des Klickbereich zum Content erfolgt nur über javascript.

Farbe gibt es mit den Klassen acco-bl(blau), acco-rd(rot), acco-or(orange), acco-ye(gelb) und acco-gn(grün), die im übergeordneten Container die Klasse acco ersetzen. Allerdings wird nur der Rand des ausgewählten Elements eingefärbt.

Info zum Footer

Den Tag <footer/> mit Klasse foot anlegen. Feststehende Footer mit zusätzlicher Klasse foot-fx
Im Footer könne andere Container eingesetzt werde z.B. <address/> allerdings ist die maximale Höhe bei einem feststehenden Footer auf 40px begrenzt, alles darüber hinaus wird ausgeblendet. Den Footer kan man mit einer Hintergrund-Klasse einfärben z.B.

Quelltext-Beispiel:
<footer class="foot foot-fx bg-gr">   <span class="h3"><a href="#sectFoot" class="t-rd">FOOTER</a></span>   <address class="c5 inl">Name Vorname, Straße Nr, 0PLZ0 Ort</address>   <b><a style="float:right;" class="btn btn-sm up" href="#" title="to top"></a></b> </footer>
